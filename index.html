Cheatsheet

Html

Html stands for Hypertext Markup Language. It is most commonly used to structure a page into a website.

Structure

<!DOCTYPE html>
<html>
<head>
<title> Example </title>
</head>
<body>
<h1> welcome to my example </h1>
</body>

css

CSS is the acronym of “Cascading Style Sheets”. CSS is a computer language for laying out and structuring web pages (HTML or XML).

Syntax

Css is written in a specific way so that the coder can write in an effiecent manner. The way it is supposed to be written is, there is a "selector", or whatever you want to style. Then there is the decleration. This is what you code to make it styled.
Ex:
.selector {
Color: red;
}
